# partum-partner
Checklist for prenatal and postpartum partner support.

## Contribution Guide

### How it works (what you can do next)

- `bundle exec jekyll serve` (if Ruby bundler + jekyll installed)
- Open `http://127.0.0.1:4000/`
- Visit:
  - `/labor/`, `/birth/`, `/postpartum/`, `/diapers/`, `/chores/`
- Edit text in Markdown; Jekyll picks up changes.

### 🔧 Optional refinements

- Add `assets/js/checklist.js` for localStorage checkbox state
- Add YAML-controlled sections (e.g., clinical intro + actions by stage)
- Add README.md user-facing overview remains as repo summary (optional)
- Add Lighthouse for accessibility style tune
