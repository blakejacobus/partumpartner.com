# Postpartum
> Anticipate needs; support MOM to support BABY. <br>
> BABY duties are opportunities to create a bond with BABY. <br>
> Create calm amidst the chaos to help MOM and BABY rest, recover, and grow.
## Goals
1. BABY is feeding every 2-3 hours.
1. BABY [diaper output](diapers.md) is tracked.
1. MOM is eating at least 3 meals and 2 snacks.
1. MOM is drinking at least 100 oz water (plus electrolytes).
1. DAD and KID(S) eating/drinking as needed.
1. DAD and/or VISITORS are completing [chores](./chores.md).
## Morning (6am-12pm)
- [ ] Feeding #1
- [ ] Meal #1
- [ ] Hydration: 40 oz (1L)
- [ ] Bathroom (Perineal care, bandages, padsicles, etc.)
- [ ] Sitz bath #1
- [ ] Feeding #2
- [ ] Snack #1
- [ ] [OPTIONAL] Nap (MOM if needed; DAD if MOM stays up or guests can babysit)
## Afternoon (12pm-6pm)
- [ ] Feeding #3
- [ ] Meal #2
- [ ] Hydration: 80 oz (2L)
- [ ] Bathroom (Perineal care, bandages, padsicles, etc.)
- [ ] Sitz bath #2
- [ ] Feeding #4
- [ ] Snack #2
- [ ] [OPTIONAL] Nap (MOM if needed; DAD if MOM stays up or guests can babysit)
## Evening (6pm-12am)
- [ ] Feeding #5
- [ ] Meal #3
- [ ] Prepare beds
- [ ] Prepare changing station (diapers, wipes, tracking sheet)
- [ ] Hydration: 120 oz (3L)
- [ ] Feeding #6
- [ ] [OPTIONAL] Snack #3 / Dessert
### MOM & BABY
- [ ] Shower/bath
- [ ] Magnesium
- [ ] Vitamins
### KID(S)
- [ ] Shower/bath
- [ ] Snack
- [ ] Pajamas
- [ ] Teeth
## Sleep (12am-6am)
- [ ] Sets 2-3h alarms for feedings/changings
- [ ] Change baby before/after feedings; update diaper tracker
- [ ] Offer water/snacks to MOM
